# HAR 2009 — Constructionist Educational Open Source SimCity

Artifacts for Don Hopkins’ **lightning talk** at [HAR 2009](https://www.har2009.org/) (Hacking at Random), July 2009, Netherlands.

## Files

| Name | Description |
|------|-------------|
| **`HAR2009Transcript.html`** | Long-form illustrated transcript with links to YouTube and many in-line figures. Open locally in a browser. |
| **`HackingAtRandomTalk.txt`** | Short prose abstract of the Micropolis educational / open-source narrative. |
| **`MicropolisHARTalk.key`** | Apple Keynote presentation bundle (slides + thumbnails). |
| **`images/`** | Photos, screenshots, slide exports, and thumbnails referenced from **`HAR2009Transcript.html`** (`src="images/..."`). |

## Legacy deployment URLs

The HTML `<meta property="og:*">` tags still mention historical **`MicropolisOnline.com/static/documentation/...`** paths. Those hosts may be gone; the **relative** `images/` links in this folder remain valid in a local checkout.

## YouTube

The transcript links a **YouTube playlist** for the lightning-talk block; if URLs rot, prefer the transcript and Keynote bundle as the canonical record.
