# Micropolis HTML manual

Static HTML version of the **Micropolis** (GPL SimCity) documentation shipped for the **OLPC / Sugar** era: player-facing introduction, walkthrough, reference, simulator overview, and supporting chapters.

## Contents

| File | Description |
|------|-------------|
| **`index.html`** | Manual home: links to all chapters, GPL section 7 addendum, OLPC support pointers (legacy URLs). |
| **`intro.html`** | Foreword, system simulation concepts, rules of city play. |
| **`tutorial.html`** | Step-by-step walkthrough; **UI descriptions target the original HyperLook / Unix client** (menus, terrain generator, etc.). Gameplay ideas still apply to modern ports. |
| **`reference.html`** | In-game tools, budgets, graphs, disasters, advisors — still useful for understanding simulation concepts. |
| **`inside.html`** | How the simulator thinks about tiles, zones, power, traffic, etc. |
| **`history.html`** | **“History Of Cities And City Planning”** by Cliff Ellis — general urbanism / planning essay bundled with the game; timeless background reading. |
| **`bibliography.html`** | Reading list tied to the manual. |
| **`credits.html`** | Credits and license text. |

## Viewing

Open **`index.html`** in a browser from a checkout (local `file:` URLs are fine). All chapter links are **relative** within this folder.

## Note on age

Some support links (wiki.laptop.org, mailto targets) and the tutorial’s **HyperLook** UI steps are **historical**. The **reference**, **inside**, and **history** chapters remain good companions when learning or teaching with MicropolisCore today.
