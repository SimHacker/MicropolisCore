# Legacy planning notes (OLPC / GTK / Tcl era)

These files captured **shipping plans** and **roadmaps** for Micropolis on **OLPC Sugar**, **Python / PyGTK**, and related stacks. They are **historical receipts**: many “TODO” items were completed or abandoned as the project moved to **WebAssembly + SvelteKit**.

## Files

| File | Description |
|------|-------------|
| **`PLAN.txt`** | OLPC feature checklist (Sugar audio, screen layout, multiplayer flags, etc.). |
| **`TODO.txt`** | Task list from the same period. |
| **`OLPC-notes.txt`** | Longer OLPC integration notes. |
| **`UserInterfacePlan.txt`** | Window / panel layout plans for the Sugar activity. |
| **`DevelopmentPlan.md`** | Don Hopkins’ essay on C++/Python split, tile engine, internationalization — starts with a disclaimer that technology moved on; still useful for **why** the core/UI boundary exists. |
| **`ROADMAP.txt`** | Old source tree map (“`src/notes`”, resources, Tcl widget notes). |
| **`DocTCL.txt`** | Tcl/Tk documentation hooks from the Unix UI line. |

If an idea here reappears in modern work, link forward to **`../../designs/`** rather than duplicating.
