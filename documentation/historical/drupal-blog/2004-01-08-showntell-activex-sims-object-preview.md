# ShowNTell ActiveX plug-in for previewing Sims objects

*Thursday, January 8, 2004*

**ShowNTell ActiveX Plug-In for Previewing Sims Objects**

ShowNTell was an ActiveX control (built on the **Transmogrifier** stack) for displaying a **live, interactive preview** of a Sims `.iff` object on a web page — rotation, zoom, name/price/description, and configurable background/grid so previews could match a site’s layout.

Goal: make it easy to distribute objects on the web and let players inspect downloads before installing them and restarting *The Sims*.
