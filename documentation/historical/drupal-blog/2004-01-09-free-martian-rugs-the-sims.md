# Free Martian rugs for The Sims

*Friday, January 9, 2004*

**Free Martian Rugs to Download for The Sims**

A batch of “Martian rugs” was posted: preview images from NASA’s web site, dragged into **RugOMatic** (with **The Sims Transmogrifier**), then published via a **Radio UserLand** workflow so objects appeared on the blog with previews and descriptions. Players could use the pie menu **Describe** action in-game to read the accompanying text.

This was framed as an early test of making creation and publishing of *Sims* objects very lightweight.
